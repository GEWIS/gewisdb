getMembersToCheck()

Member.php : 37

Get a list of members whose membership should be checked against the TU/e student administration database.

public getMembersToCheck(int $limit) : array<string|int, Member>

This utilises a hidden column that fixes the sorting of records that have a lastCheckedOn of null. Postgres defaults to sorting null after everything else when using ASC. It does support NULLS FIRST after an ORDER BY to reverse this, however, Doctrine ORM does not support this.

The hidden column contains either 0 or 1, when lastCheckedOn is null or not (respectively). We sort on this column first in ascending order, causing all records having null for lastCheckedOn to appear first.
